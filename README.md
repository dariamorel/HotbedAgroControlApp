# HotbedAgroControlApp
Данное приложение создано для удаленного управления устройством HotbedAgroControl: https://github.com/woronin/HotbedAgroControl.

На данном этапе приложение подключено к конкретному устройству. В последующих версиях появится возможность добавления собственного устройства.

#### Установка приложения
Запуск из Android Studio путем клонирования репозитория или установка напрямую через файл app.apk.

#### Реализовано:
- Подключение к Mosquitto по WIFI и протоколу MQTT для управления устройством.
- Отображение значений датчиков и возможность взаимодействия с элементами управления.
- Кеширование данных полученных с датчиков в базу данных.
- Отображение графиков с историей изменения данных.

#### Используемые технологии:
- Сетевые запросы: Mqtt, Mosquitto
- Хранение данных: Room, SharedPreferences
- Архитектура: Clean Architecture + MVVM
- UI: Jetpack Compose
- Графики: Compose Charts
- DI: Hilt
- Асинхронность: Coroutine
